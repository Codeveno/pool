      HTML
      HTML (HyperText Markup Language) is the standard language
      used to create and structure web pages. It uses elements and
      tags to define content such as text, images, links, and
      multimedia.
      CSS
      CSS (Cascading Style Sheets) styles and enhances the
      appearance of web pages by controlling layout, colors, fonts,
      and responsiveness.
      JavaScript
      JavaScript (JS) is a programming language that adds
      interactivity, dynamic content, and functionality to websites.
